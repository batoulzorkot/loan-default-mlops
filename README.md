# Loan Default Prediction — MLOps Project

## Description
Modèle de prédiction de défaut de prêt bancaire avec pipeline MLOps complet.

## Stack technique
- Python 3.10
- Scikit-learn
- MLflow
- Streamlit
- GitHub Actions (CI/CD)

## Installation
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Lancer l'app
```bash
streamlit run app/app.py
```
